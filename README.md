# ⚔️ DuelCom

DuelCom is a real-time competitive coding platform where users solve bite-sized algorithmic problems in 1v1 duels under time pressure. It is designed to make coding practice fast, engaging, and competitive.

---

## 🚀 Features

- ⏱️ **Real-time Coding Duels**  
  Compete head-to-head with another user on the same problem within a fixed time limit.

- 🧠 **Curated Problem Set**  
  Easy-to-medium algorithmic problems focused on logic, data structures, and pattern recognition.

- 🏆 **Leaderboard System**  
  Track wins, rankings, and performance over time.

- 👤 **User Profiles**  
  View duel history, stats, and progress.

- 🔔 **Notifications**  
  Get notified for duel invites and results.

---

## 🛠️ Tech Stack

### Frontend
- React.js  
- Tailwind CSS  
- JavaScript / TypeScript  

### Backend
- Node.js  
- Express.js  
- REST APIs  

### Database & Auth
- Firebase (Auth + Firestore)

---

## 🔌 API Integration

- All backend APIs are ready and exposed via REST.
- Frontend communicates with backend for:
  - Duel creation & matchmaking
  - Problem fetching
  - Submission evaluation
  - Result storage
  - Leaderboard updates
